    SensorbergSDK Reference

    Introduction

    SensorbergSDK implements support for accessing beacons that are managed via the Sensorberg Beacon Management Platform.

    Prerequisites

    1. Before you integrate SensorbergSDK into your own app, you should create an account at the Sensorberg Beacon Management Platform if you haven’t already.
    2. We also assume that you already have a project in Xcode and that this project is opened in Xcode.
    3. The SDK supports iOS 7.0 or newer.
